## 1. HTTP 헤더의 개요
  + 헤더 분류
    + General 헤더: 메시지 전체에 적용되는 정보
    + Request 헤더: 요청 정보
    + Response 헤더: 응답 정보
    + Entity 헤더: 엔티티 바디 정보 ex) Content-Type
